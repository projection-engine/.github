# Projection engine

The **Projection Engine** is a 3D graphics engine written from ground up to be multi-platform and easy to use.


## Download

Download the latest version: [v2.4.0-alpha](https://github.com/projection-engine/editor/releases/tag/v2.4.0-Alpha)
